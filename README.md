# Futbol Magaly

Futbol Magaly is a soccer game-day and training web app built with React, Vite, and Tailwind CSS.

The app includes a jersey roulette picker, a juggling dice game with scoreboard, and an interactive soccer field challenge page. <img width="1667" height="997" alt="Screenshot 2026-04-21 at 5 39 04 PM" src="https://github.com/user-attachments/assets/b65fbb9e-db5b-4e5d-9e65-cdc85e208674" />


* Features

- Jersey roulette wheel
- Import jersey images
- Crop uploaded jersey pictures
- Save jerseys in the browser
- Spin animation with sound effects
- Winner celebration with confetti/spaghetti effect
- Background music upload
- Soccer dice challenge page
- Dice landing sound and screen shake
- Player scoreboard with saved scores
- Target score and winner celebration
- Interactive soccer field challenge page
- Random field challenge picker
- Field challenge scoreboard
- Firebase Hosting setup
